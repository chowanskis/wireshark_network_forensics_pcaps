# wireshark_network_forensics_pcaps
PCAP files with detailed analysis for learning network traffic inspection and cybersecurity.

A collection of PCAP files for network traffic analysis, along with detailed explanations and step-by-step investigations.
This repository is designed for learning and practicing with Wireshark and other network analysis tools. Each sample includes real-world scenarios such as normal traffic, suspicious behavior, and potential security incidents.
The goal is to help learners understand how to analyze network traffic, identify anomalies, and develop practical skills useful for SOC analysts, network engineers, and cybersecurity professionals.
